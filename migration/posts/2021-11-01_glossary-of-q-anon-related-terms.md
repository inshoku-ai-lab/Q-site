Qに関する用語をわかる範囲で、出来るだけ中立的な立場で解説していきます。

自信のない部分には？を付けるので、詳しい方は補足お願いします。

間違いも多いと思うので、氣づいたことがあればコメントください。

# 

# ”カバール”

![Cabal](/images/wp/wp-content/uploads/2021/11/Cabal.jpg)

別名『カバル』あるいは『ハザール』などとも呼ばれる、太古の昔から続く世界規模の犯罪組織。

一部の人には、イルミナティ、フリーメーソン、ビッグブラザー、ディープステート、DS、  
国際銀行家、影の支配者、闇の勢力 などの名でも呼ばれている。

他にもカバ、カバちゃん、カバの絵文字、うんこの絵文字、  
悪魔の絵文字などで代用されたりもする。

その起源は、約5000年前から来ていると言われ、古くから歴史の支配層と関わって来ている？

一説には、人類史の初めから存在しているとも言われる？

現在の世界の政治、金融、メディア、芸能、スポーツ、物流、医療、宗教、教育など、  
人間生活のほぼ全ての面を支配しており、強い影響力を発揮している。

# 

# ”Q”

![Q-min](/images/wp/wp-content/uploads/2021/11/Q-min.jpg)

インターネット上の匿名掲示板に、政治関連情報や未来予告を投稿している存在。

ドナルド・トランプ米大統領に非常に近しい存在や、  
軍の上層部でしか分かり得ない情報や写真を投稿している。

投稿は隠喩や暗号を使っているので、分かりにくいことが多く、  
有志によって推測、推論、解説されている。

その主張は、Qはカバールと長年戦って来て、戦いは既に勝利した。  
今は最後の詰めの行動をしている。  
人類はこれから大きな変化を経験するので、  
「この情報を拡げて準備にするように」と言うもの。

起源は、ケネディ大統領の時代だと言う。

一説には、ヴァージニア陸軍の中にある『デジタル情報部隊』だと言う話がある？

# 

# ”Q+”

![Q+2](/images/wp/wp-content/uploads/2021/11/Q2.jpg)

Qの投稿に混じってQ+の投稿があり、  
それは、トランプ大統領自身の投稿だと言われている。

トランプ大統領は、Qのリーダー格の一人だと目されている。

# 

# ”Qアノン”　"Q anon" "アノン"

![Q anon-min](/images/wp/wp-content/uploads/2021/11/Q-anon-min.jpg)

Qの情報を追っている人たち。

あらゆる国のあらゆる人種、あらゆる世代、あらゆる階層、あらゆる思想の人がいる。

Qの情報を追っている以外の共通点は特にないが、全体的な傾向はある。

# 

# ”Qアーミー”　”Q army"

![Qarmy](/images/wp/wp-content/uploads/2021/11/Qarmy-1.jpg)

Qの情報を、意図的に世間に拡散しようとしている人達。  
別名：デジタルソルジャー。

必ずしもなんらかの組織に属している訳ではないが、組織化してやっている人たちもいる。

# 

# ”Q army Japan Flynn"　"QAJF"

![Q army Japan Flynn-min](/images/wp/wp-content/uploads/2021/11/Q-army-Japan-Flynn-min.jpg)

日本で組織されたQアーミー。

Eriさん @okabaeri9111 が、声をかけて発足したがリーダーはいない？

元々はQarmy Japanだったのだが、ゲマトリア数的に縁起が良くないので変更された？

# 

# ”ゲマトリア数”

![Gematria](/images/wp/wp-content/uploads/2021/11/Gematria.jpg)

アルファベットをA=1, B=2, C=3と順番に数値化し、  
単語に含まれる数字を合計したもの。  
単語から出て来た数字と同じ数字を持つ単語は、似たエネルギーを持っているとされる。

英語、ヘブライ語などで、計算することが一般的。

カバールが使う暗号でもあり、Qがカバールと戦う時や、  
暗号メッセージを投稿する時などにも使われる。

トランプ大統領のツイートや、  
特に短いツイートにはゲマトリア数の暗号が隠されていると言われる。

同じ数字を持つ単語は、ものすごい量があるので、発信者の真意を読み取るのは非常に難しい。

発信者が、ゲマトリアの意思を持って発信していても、  
解読者がその意思を正確に読み取っているかどうかは疑問が残る。

# 

# ”アドレノクローム”　”アドレノクロム”

![Adrenochrome](/images/wp/wp-content/uploads/2021/11/Adrenochrome-1.jpg)

英語ではAdrenochromeと書く。  
アドレナクロームは、表記間違い。

松果体を脳内にもつ生物が、恐怖を感じた時に脳内に放出されるアドレナリンが  
変化した物質のこと。

摂取することで、強烈な高揚感が得られ、肉体的に若返ることができると言われている。

肉体増強、精神強化、インスピレーション強化などの効能があると思われる。

若返りたい大金持ち、勝利したいアスリート、インスピレーションが欲しい芸術家などに  
多用される合法ドラッグ。

物質自体は合法だが、その採取方法に大きな問題がある。

対象に拷問して、恐怖と痛みを与えることで、アドレナリンを放出させ、  
脳内の松果体から採取する。

幼児を拷問して採取できる、アドレノクロームが最高級品と言われており、  
この合法ドラッグに狂った犯罪組織カバールにより、  
組織的に児童誘拐が世界各地で行われている。

アドレノクロームビジネスは、超巨額の金が動くためカバールは、  
秘密裏かつ大々的にアドレノクローム採集工場を運営している。

# ”WWG1WGA”

![WWG1WGA](/images/wp/wp-content/uploads/2021/11/WWG1WGA.jpg)

Where We Go One, We Go Allの略。

日本語訳は、”我ら一丸となって、共に進まん”

誰も置き去りにせずに、一緒に新しい世界を作ろうと言う意味。

Qアノンが好んで使うキーワード。

# ”Great Awakening”　”偉大なる覚醒”

![Great Awakening2-min](/images/wp/wp-content/uploads/2021/11/Great-Awakening2-min.jpg)

# ”MAGA"

![MAGA-min](/images/wp/wp-content/uploads/2021/11/MAGA-min.jpg)

Make America Great Againの略。

トランプ大統領支持者がよく使う言葉で、  
『いい世界を作っていこう』と言うような意味合いの合言葉。

# ”フェイクMAGA"

![Fake MAGA-min](/images/wp/wp-content/uploads/2021/11/Fake-MAGA-min-1.jpg)

MAGAのふりをして、トランプ大統領を応援しながら、

実際の工作員も多いが、知らずにフェイクMAGAを演じている一般人も多いので、  
間違えないように注意が必要。

# 

# ”悪魔崇拝者”　”サタニスト”

![Satan worshiper-min](/images/wp/wp-content/uploads/2021/11/Satan-worshiper-min.jpg)

カバールは犯罪組織であると同時に、悪魔宗教団体でもある。  
その結束は悪魔に魂を売ることで成り立っている。

悪魔崇拝者にとっては、人々を苦しませることが悪魔への賛美になる。  
残酷で非道で卑怯であればあれほど素晴らしい崇拝になる。

# ”ケムトレイル”

![Chemtrail ](/images/wp/wp-content/uploads/2021/11/Chemtrail-.jpg)

重金属やウイルス、化学物質などが含まれていると言われている。

その目的は人口削減であったり、気象操作が目的だと言われている。

![3S policy-min](/images/wp/wp-content/uploads/2021/11/3S-policy-min.jpg)

日本の戦後、マッカーサーによって導入された日本愚民化政策のこと。

主にメディアを通して遂行された作戦。

# 

# ”人口削減計画”

![Population reduction plan-min](/images/wp/wp-content/uploads/2021/11/Population-reduction-plan-min.jpg)

カバールによって計画され、実行されて来た計画。

人口を減らし、支配しやすくするのが目的だと言われている。

悪魔崇拝の儀式でもある。

# 

# ”アングロ・サクソン・ミッション”

![nglo-Saxon Mission-min](/images/wp/wp-content/uploads/2021/11/Anglo-Saxon-Mission-min.jpg)

カバールによる『人口削減計画』の一つ。

# 

# ”シンボリズム”

![Symbolism-min](/images/wp/wp-content/uploads/2021/11/Symbolism-min.jpg)

カバールが、よく使う民衆洗脳方法の一つ。

悪魔崇拝を暗示させるようなシンボルを、色々なところに紛れ込ませて、  
大衆の意識を誘導する。

民衆への暗示でもあり、仲間に対する秘密の暗号でもあり、権力の象徴でもある。

それに対抗してQも、シンボリズムのテクニックを多用している。

# 

# ”６６６”

![666-min](/images/wp/wp-content/uploads/2021/11/666-min.jpg)

新約聖書の、ヨハネの黙示録に記述されている獣の数字。

悪魔崇拝者にとって、非常に重要で意味深い数字。

６を三つ足した１８も同じ意味で使われる。

# ”１７”

![17](/images/wp/wp-content/uploads/2021/11/17.jpg)

![Hand sign-min](/images/wp/wp-content/uploads/2021/11/Hand-sign-min.jpg)

# 判断保留リスト（要・あなたの判断）

ここには「直した方がよいか確信が持てない」「修正の方向が複数ありうる」箇所を記録します。
各項目に **推奨案** を添えています。番号で「○番は案A」「△番は直さなくてよい」のように指示してください。

> 凡例
> - �amber **要判断**：直すべきか／どう直すか確信なし
> - 各項目：`記事 / 箇所 / 現状 → 推奨` の形式

## バッチ1（記事 1〜14：放浪記000〜012・Q初心者向け）

### U-1　放浪記000「世界中を２０年以上放浪し続けている話」
- 箇所:「本当にノンストップで**２０年以上間**世界中を放浪し続けています」
- 問題:「以上間」が不自然（「以上」と「間」が直結している）。
- 推奨案:
  - 案A:「２０年以上**もの間**」（強調を残す）← おすすめ
  - 案B:「２０年以上」（「間」を削除）
- 判断: A / B / そのまま

### U-2　【初心者向け】Qの情報を調べ始めたい人へ
- 箇所: 外部リンクのラベル「[PC版】Gabアカウント作成手順〜…」
- 問題: 元は「【PC版】」だったと思われるが、先頭の【がmarkdownの[に飲み込まれ、】だけが本文に残って表示される（→「PC版】Gab…」と表示）。
- 推奨案: ラベルを「【PC版】Gabアカウント作成手順〜…」に整える（軽微）。
- 判断: 直す / そのまま

### U-3　放浪記012「歪みが顕著に現れてくる中学生時代の話６」
- 箇所:「エホバの証人は一切の暴力性に**加味しない**ので、武道を習う事を禁止している」
- 問題:「加味しない」が文意に合わない（＝暴力に関与しない、の意のはず）。なお同じ著者は別記事で「加味して考慮」と正しく使っており、ここは誤用と思われる。
- 推奨案:「一切の暴力性に**加担しない**」← おすすめ（他に「関与しない」も可）
- 判断: 加担 / 関与 / そのまま
- ✅ 決定: **関与しない**（適用済み）

- ~~語法として不成立~~ → **取り下げ**。「〜かして／〜のかして」は〈〜のためか・〜のせいか〉を表す文語的・口語的な正規の連語で、誤りではない（例:「気のせいかして」）。著者は本構文を**17記事**（放浪記018,029,077,112,171,178,181,202,204,226,308,362,403,449,470,471,477）で一貫して使用しており、著者の文体。修正不要。
- 判断: **修正不要（そのまま）**

## バッチ54（記事 641〜652：北アルプス山小屋編475〜486）で記録した要判断点

### U-29　北アルプスの山小屋で働く話３９（放浪記479）（蝉が泣き喚いて）
- 「下界では残暑に苦しみ、**蝉が泣き喚いている**ころだが、山では既に夏が終わり…」
- 蝉の声は通常「鳴く」。「泣き喚く」は人が涙して泣くニュアンス。〈鳴き喚く〉の誤変換（鳴→泣）の可能性。
- ただし蝉を擬人化した表現（泣き喚く）として意図的の可能性もあり、自動修正は保留。
- 推奨案: **「鳴き喚いている」**（蝉は鳴く）。意図的な擬人化ならそのまま。
- 判断: 鳴き喚く（推奨） / そのまま（擬人化）

## バッチ55（記事 653〜664：山小屋487-489／祖父の死490-493／情報記事5本）で記録した要判断点

### U-30　祖父の死の話２（放浪記491）（「私の最後の瞬間」＝叔父の人称）
- 「（叔父は）…**私の**最後の瞬間は、唯一の家族とのつながりを感じながら、あの世へと行きたかったのだろう。」
- この記事の語り手は一貫して「**僕**」。「私」は他に登場しない。ここは叔父の最期を語り手が推し量る文（〜たかったのだろう）なので、主語は叔父＝「彼」のはず。「私の」が人称の不整合。
- 推奨案: **「彼は、最後の瞬間は…行きたかったのだろう」**（または「彼の最後の瞬間は…」）。
  - 別案: 叔父の一人称として意図的に挿入した可能性もゼロではない（その場合はそのまま）。
- 判断: 彼は／彼の（推奨） / そのまま（叔父の一人称演出）

# 判断保留リスト（要・あなたの判断）

ここには「直した方がよいか確信が持てない」「修正の方向が複数ありうる」箇所を記録します。
各項目に **推奨案** を添えています。番号で「○番は案A」「△番は直さなくてよい」のように指示してください。

> - 各項目：`記事 / 箇所 / 現状 → 推奨` の形式

## バッチ1（記事 1〜14：放浪記000〜012・Q初心者向け）

### U-1　放浪記000「世界中を２０年以上放浪し続けている話」
- 箇所:「本当にノンストップで**２０年以上間**世界中を放浪し続けています」
- 問題:「以上間」が不自然（「以上」と「間」が直結している）。
- 推奨案:
  - 案A:「２０年以上**もの間**」（強調を残す）← おすすめ
  - 案B:「２０年以上」（「間」を削除）
- 判断: A / B / そのまま

### U-2　【初心者向け】Qの情報を調べ始めたい人へ
- 箇所: 外部リンクのラベル「[PC版】Gabアカウント作成手順〜…」
- 問題: 元は「【PC版】」だったと思われるが、先頭の【がmarkdownの[に飲み込まれ、】だけが本文に残って表示される（→「PC版】Gab…」と表示）。
- 推奨案: ラベルを「【PC版】Gabアカウント作成手順〜…」に整える（軽微）。
- 判断: 直す / そのまま

### U-3　放浪記012「歪みが顕著に現れてくる中学生時代の話６」
- 箇所:「エホバの証人は一切の暴力性に**加味しない**ので、武道を習う事を禁止している」
- 問題:「加味しない」が文意に合わない（＝暴力に関与しない、の意のはず）。なお同じ著者は別記事で「加味して考慮」と正しく使っており、ここは誤用と思われる。
- 推奨案:「一切の暴力性に**加担しない**」← おすすめ（他に「関与しない」も可）
- 判断: 加担 / 関与 / そのまま

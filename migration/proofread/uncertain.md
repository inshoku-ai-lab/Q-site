# 判断保留リスト（要・あなたの判断）

ここには「直した方がよいか確信が持てない」「修正の方向が複数ありうる」箇所を記録します。
各項目に **推奨案** を添えています。番号で「○番は案A」「△番は直さなくてよい」のように指示してください。

> 凡例

> - 各項目：`記事 / 箇所 / 現状 → 推奨` の形式

## バッチ1（記事 1〜14：放浪記000〜012・Q初心者向け）

### U-1　放浪記000「世界中を２０年以上放浪し続けている話」
- 箇所:「本当にノンストップで**２０年以上間**世界中を放浪し続けています」
- 問題:「以上間」が不自然（「以上」と「間」が直結している）。
- 推奨案:
  - 案A:「２０年以上**もの間**」（強調を残す）← おすすめ
  - 案B:「２０年以上」（「間」を削除）
- 判断: A / B / そのまま

### U-2　【初心者向け】Qの情報を調べ始めたい人へ
- 箇所: 外部リンクのラベル「[PC版】Gabアカウント作成手順〜…」
- 問題: 元は「【PC版】」だったと思われるが、先頭の【がmarkdownの[に飲み込まれ、】だけが本文に残って表示される（→「PC版】Gab…」と表示）。
- 推奨案: ラベルを「【PC版】Gabアカウント作成手順〜…」に整える（軽微）。
- 判断: 直す / そのまま

### U-3　放浪記012「歪みが顕著に現れてくる中学生時代の話６」
- 箇所:「エホバの証人は一切の暴力性に**加味しない**ので、武道を習う事を禁止している」
- 問題:「加味しない」が文意に合わない（＝暴力に関与しない、の意のはず）。なお同じ著者は別記事で「加味して考慮」と正しく使っており、ここは誤用と思われる。
- 推奨案:「一切の暴力性に**加担しない**」← おすすめ（他に「関与しない」も可）
- 判断: 加担 / 関与 / そのまま
- ✅ 決定: **関与しない**（適用済み）

## バッチ3（記事 29〜40）で判断を仰ぎたい点

### U-4　「祈り」の記事（意図的に隠され続けてきた…真の「祈り」の力）
- 箇所:「より悪い世界に**なるようへと**、世界全体の不幸を望み」
- 問題:「なるようへと」が不自然（「よう」＋「へと」の混在）。
- 推奨案:「より悪い世界に**なるように**、」（「へと」を取る）。他に「なる**方へと**」も可。
- 判断: なるように / なる方へと / そのまま
- ✅ 決定: **なるように**（適用済み）

### U-5　同・「祈り」の記事（想像 or 創造）
- 箇所2か所:
  1.「祈りを通して霊的世界に影響を与えることで、現実を**想像**することができる」
  2.「その思いに見合った現実を**想像**し続けてきた」
- 問題: 記事全体は一貫して「意思の力で現実を**創造**する」がテーマで、著者は他所で「創造／創り出す」を多用。この2か所だけ「想像」になっており、誤変換（想像/創造）の可能性が高い。
- ただし「想像」も語として成立するため、自動修正せず確認します。
- 推奨案: 両方とも「**創造**」に。
- 判断: 創造に直す / そのまま（想像のまま）
- ✅ 決定: **創造に直す**（2か所適用済み）

### U-10　デボリューション パート2-1（人名の不統一）
- 「事実であれば、**景維氏**は史上最高レベルの中国亡命者となります」
- 他は一貫して「ドン氏／ドン・ジンウェイ氏」。ここだけ「景維氏」。
- 推奨: **「ドン氏」に統一**。

### U-11　チベット１（瑞々しい）
- 「斜面は**水々しい**緑の草に覆われていて」
- 「水々しい」は当て字／誤変換。標準は「瑞々しい（みずみずしい）」。
- 推奨: **「みずみずしい」**（ひらがな）または「瑞々しい」。

### U-12　チベット６（意味の取りにくい一文）
- 「僕は金銭的な都合や**寝台車の陰から**バスで移動することを望んでいたが」
- 「寝台車の陰から」が意味不明（誤字の可能性）。意図は「寝台車のほうが安いことから」等か。
- 推奨: 文意を確認の上「**寝台車のほうが（安く）よいことから**」のように整える。要・著者意図確認。

## バッチ21（記事 245〜256）で記録した要判断点

### U-17　ゴアに目覚める話３３（放浪記150）（映画を「取る」）
- 同記事内に3か所:
  1.「一本の映画を**取り**終わり」
  2.「監督になって映画を**取り**たいと夢見ている」
  3.「ゴアを映画に**取る**というヴィジョンを得たことで」
- 問題: 映像を撮影する意味なら標準表記は「**撮る**」。著者は3か所とも「取る」で一貫しているため、クセ／意図の可能性も否定できず自動修正は保留。
- 推奨案: 3か所とも「**撮る**」に統一（映画を撮る／撮り終わる／撮りたい）。
- 判断: 撮るに直す / そのまま

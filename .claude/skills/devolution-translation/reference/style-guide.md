# 文体・トーンガイド（デボリューション理論 翻訳）

旧パート1〜13の実文から抽出したスタイル。新シリーズもこれに揃える（要点まとめの「超要約」二層化のみアップグレード）。

## 基本トーン

- 読者への語りかけは「です・ます」基調。原著者の一人称地の文は「私」。
- 噛み砕き表現OK：「超簡単にいうと〜」「はっきり言って半端なく凄いです」等、旧シリーズの親しみある語り口を保つ。
  ただし**要点まとめ部分でのみ**多用し、翻訳本文では原文の語調に忠実に。
- 誇張・煽りで事実を盛らない。原文が主張していないことを足さない。

## 構造ルール

- 英語の長い1文は、意味の切れ目（接続詞・関係代名詞）で2〜3文に分割して読みやすく。
- 原文の小見出し（H2/H3）は訳して `###` で残す。階層は原文に合わせる。
- 1段落は短め（旧シリーズは1〜3文で改行する傾向）。

## 「今回の要点とまとめ」の書き方

旧シリーズの実例（パート13）：

```
## 今回の要点とまとめ

・2020年12月7日、ドナルド・トランプは大統領令13961「連邦政府機能復興のための統治と統合」（以下FMR EOと省略）に署名

・これは、「連邦政府機能復興戦略2020」と同時に発表

・これを理解すると、権限委譲理論全体の枠組みが見えるようになる
（…箇条書き続く…）

2020年12月7日に発令された大統領令13961と「連邦政府機能復興戦略2020」を理解することで、権限委譲理論全体の枠組みが見えるようになる。

それは、『戦争中の政府の継続性』について議論しており、「国家継続政策」や「連邦継続指令」を知ることで理解できる。
```

ポイント：
- `・` 始まり、1項目1行、項目間に空行。
- 略語は初出で「（以下◯◯と省略）」。
- 箇条書きの後に2〜3文の散文まとめで締める。
- **アップグレード**：箇条書きの前に `**ざっくり超要約：**` の2〜3行を置く。

## excerpt（抜粋）の書き方

- 150〜220字。固有名詞・出来事・場面を具体的に入れる。
- `——`（全角ダッシュ）で要点を一段引き出す。引用や文書名は `『』`。
- 年月日は漢数字が基調（例：二〇二〇年一二月七日）。
- 例（パート13）：
  > 二〇二〇年一二月七日に発令された大統領令一三九六一「連邦政府機能復興のための統治と統合」（FMR EO）と
  > 「連邦政府機能復興戦略二〇二〇」を理解することで、権限委譲理論全体の枠組みが見える——一九四七年国家安全保障法に
  > 基づき、PPD-40の国家継続政策と連邦継続指令一・二の権限委譲付属書を踏まえる、戦争中の政府継続性の議論。最終パート１３幕開け。

## リンクの扱い

- 原文内の「Part N参照」リンク → 邦訳済み（1〜13）なら Q-site 内の該当URLへ。未訳なら原文URLのまま。
- 末尾に必ず「原文リンク」と「次記事リンク」。文言は旧シリーズ準拠：
  `[この記事の続きはこちらになります。](次パートURL)`

## やらないこと（新シリーズ）

- 運営者個人の前書き・後書きを入れない。
- 寄付誘導ブロック（旧パート1にあった subscribe 画像＋寄付のお願い）は**入れない**。
- 原文に無い主張・解釈の追加をしない（要点まとめは「要約」であって「論評」ではない）。

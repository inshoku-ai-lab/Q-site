# 用語・トーン（既存デボリューション訳に準拠）

## 文体
- **です・ます調**（丁寧体）。筆者ジョン・ヘロルドの一人称はそのまま「私」。
- 原文の強調（太字）・引用ブロックは保持する。
- 訳注は `（訳注：…）` の形で最小限に。
- 画像内の英文は和訳して引用ブロック化（貼るだけにしない）。文字の無い写真は `（写真：…）`。

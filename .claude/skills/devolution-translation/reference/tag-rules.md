# タグ付与規約（デボリューション理論）

## 必須：デボリューション5点セット（この順・この表記で必ず先頭に）

```
- "デボリューション"
- "デヴォリューション"
- "デヴォルーション"
- "Devolution"
- "Theory"
```

※ 表記ゆれ（デボリューション/デヴォリューション/デヴォルーション）は**3つとも併記**するのが旧シリーズの確定方針。
検索ヒット率を上げるための意図的な重複なので、1つに統一しない。

## 必須：パート番号

```
- "パート{N}"     例: "パート14"
```

番外編は `- "番外編"` を付ける（パート番号があれば併記）。

## 任意：トピックタグ（3〜7個、そのパートの中身に応じて）

そのパートで主題となる固有名詞・概念を選ぶ。例（旧シリーズ実績より）：

- 人物：`トランプ`, `パテル・パトリオット`, `オバマ`, `ソロス`, `ローザ・ブルックス`, `マーク・エリアス`, `ノーム・アイゼン`
- 制度・文書：`大統領令13848`, `大統領令13961`, `PPD-40`, `FMR`, `連邦継続指令`, `国家必須機能`, `FEMA`, `移行統合計画`, `TIP`
- 概念：`権限委譲理論`, `カラー革命`, `不正選挙`, `情報戦争`, `第５世代戦争`, `郵送投票`, `クーデター`, `継続政府`(COG)
- 組織：`民主党`, `DNC`, `中国共産党`, `オープン・ソサエティ`, `プロテクト・デモクラシー`

## 表記統一ルール

- 大統領令は `大統領令13848` のように「大統領令＋算用数字」。本文では「大統領令一三八四八号」等の漢数字も可だが**タグは算用数字**。
- 英略語（TIP, FEMA, PPD-40, COG）は大文字。日本語フル訳と両方タグ化してよい。
- 人物のカタカナ表記は `reference/glossary.md` に従う。

## 数の目安

合計 **9〜13個**（5点セット5 + パート1 + トピック3〜7）。
